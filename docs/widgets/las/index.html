Introduction to Local Alignment Score

What is Local Alignment Score (LAS)? LAS measures the average semantic quality of matched chunk pairs between reference and generated texts. Unlike Global Alignment Score (GAS) which captures overall thematic similarity, LAS quantifies fine-grained semantic coherence, detecting local inconsistencies and missing details at the chunk level.

Measures average semantic quality of matched chunk pairs. Quantifies fine-grained semantic coherence, detecting local inconsistencies and missing details.

Local Alignment Score Algorithm

Algorithm Overview: The LAS algorithm follows 3 steps to measure fine-grained semantic coherence between matched chunk pairs:

Step 1: Precision LAS Calculation

We average the similarity scores of precision best matches that were acquired after running the best matching algorithm.

$\text{precision\_sim\_values} = \{s_{i,j} \mid (i,j) \in \text{precision\_best\_matches}\}$

$\text{precision\_las} = \frac{1}{|\text{precision\_sim\_values}|} \sum_{s \in \text{precision\_sim\_values}} s$

where $s_{i,j}$ represents the similarity between generated chunk $i$ and reference chunk $j$.

Step 2: Recall LAS Calculation

We average the similarity scores of recall best matches that were acquired after running the best matching algorithm.

$\text{recall\_sim\_values} = \{s_{i,j} \mid (i,j) \in \text{recall\_best\_matches}\}$

$\text{recall\_las} = \frac{1}{|\text{recall\_sim\_values}|} \sum_{s \in \text{recall\_sim\_values}} s$

where $s_{i,j}$ represents the similarity between reference chunk $i$ and generated chunk $j$.

Step 3: F1 Score Combination

Combine precision and recall LAS using harmonic mean to create a balanced measure that captures both directional alignment qualities.

$\text{f1\_las} = \frac{2 \times \text{precision\_las} \times \text{recall\_las}}{\text{precision\_las} + \text{recall\_las}}$

$\text{LAS} = \text{f1\_las}$

The harmonic mean ensures balanced evaluation and prevents dominance by either precision or recall components.

      Testing Guide

      Getting Started

      Understanding Local Alignment Score

      This guide will help you understand how LAS works by exploring different scenarios and their impact on the semantic quality scores.

      • LAS measures the average semantic quality of matched chunk pairs
      • Precision LAS evaluates generated text quality against reference text
      • Recall LAS evaluates reference text coverage by generated text
      • Close this guide and use the Test Scenarios to experiment with different configurations

      Testing Omissions

      Impact on Recall

      When content is omitted from the generated text, recall is primarily affected as reference chunks lose their matches.

      • Click "Omissions" to simulate missing content in generated text
      • Notice how Recall LAS decreases significantly
      • Precision LAS may remain relatively stable
      • Observe the visualization showing fewer generated chunks available for matching

      Close this guide and click "Omissions" to see this effect.

      Testing Hallucinations

      Impact on Precision

      When hallucinated content appears in generated text, precision is primarily affected as generated chunks struggle to find good matches.

      • Click "Hallucination" to simulate extra content in generated text
      • Notice how Precision LAS decreases significantly
      • Recall LAS may remain relatively stable
      • Generated chunks are forced to match with fewer available reference chunks

      Close this guide and click "Hallucination" to see this pattern.

      Testing Misalignment

      LAS Insensitivity to Order

      Unlike some metrics, LAS is not sensitive to misalignment since it uses best-match pairing regardless of chunk order.

      • Click "Misalignment" to scramble the order of generated chunks
      • Notice that LAS scores remain relatively stable
      • The best-match algorithm finds optimal pairings regardless of order
      • This demonstrates LAS's focus on semantic content over structural alignment

      Close this guide and click "Misalignment" to observe this behavior.

      Manual Chunk Management

      Progressive Quality Degradation

      Manually activate/deactivate chunks to understand how LAS responds to different content availability scenarios.

      • Close this guide and click individual chunks to toggle their state
      • Try progressively disabling chunks and watch LAS scores decrease
      • Notice that LAS never reaches exactly 0.000
      • This is because averaging means even poor matches contribute some similarity
      • Experiment with disabling all but one chunk to see extreme averaging effects

      Key Insights

      Understanding LAS Behavior

      After exploring the different scenarios, you should understand these core LAS characteristics:

      • LAS measures fine-grained semantic quality through chunk-level matching
      • Omissions primarily hurt recall; hallucinations primarily hurt precision
      • LAS is insensitive to chunk order due to best-match pairing
      • LAS rarely reaches 0 due to averaging - even poor matches contribute
      • The metric provides balanced evaluation through F1 combination of precision and recall
